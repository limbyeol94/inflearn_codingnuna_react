### Props
- react에서 부모 컴포넌트가 자식컴포넌트에게 데이터나 함수를 전달하는 방법
- props는 컴포넌트 속성처럼 전달되며, 자식컴포넌트 내에서 `props.속성이름`형태로 접근할 수 있다

### onClick (이벤트 핸들러)
- React렌터링 시 함수가 즉시 실행되는 것을 막기위해 `()=>myFunction`와 같이 화살표 함수로 감싼다.
- myFunction() 을 직접넘기면, 렌디링 될 때 즉시 함수가 실행되어버린다, 화살표 함수로 감싸면 이벤트 발생 시점에만 실행된다.

### useState
- 컴포넌트 상태를 관리하며, 상태 값이 변경될 때마다 해당 컴포넌트를 다시 렌더링 하게 만드는 Hook
- UI에 영향을 주는 동적인 데이터이며 Hook을 사용하여 상태로 관라한다.
- 상태 값과 그 값을 업데이트하는 함수를 반환하며, 업데이트 시 리렌더링을 유발

### 초기값 null or undefined 일 수 있는 데이터의 렌더링
- `&&`논리 연산자 등을 사용한 조건부 렌더링을 활용한다
- data가 존재할 때만 뒤의 코드를 실행하여 에러를 방지할 수 있다.

### 무작위로 선택하려는 java Script 메서드의 활용
- Math.random(), Object.keys()를 조합한다
- Math.random()로 무작위(랜덤한) 숫자를 얻고
- Object.keys()로 객체의 키 배열을 만든 뒤 무작위 숫자를 이용해 배열의 인덱스를 선택하여 해당 아이템을 가져오는 방식을 사용함
